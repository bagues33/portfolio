My Project

Not A Dead End

NOT A DEAD END merupakan project video game online labirin yang merupakan ide oleh seniman Fajar Riyanto ini merupakan simulasi perjalanan seorang pengungsi mencari suaka di negara lain untuk mencari kehidupan yang lebih baik (lebih manusiawi). Not A Dead End dihadirkan dalam bentuk “dingdong”, atau mesin arcade, sebagai bagian dari Jogja Biennale ke-16 bertajuk “Roots <> Roots” di Yogyakarta (6 Oktober–14 Nopember 2021). Project ini menggunakan teknologi Css, Html, Javascript, dan PHP serta MySql sebagai databasenya.

Sistem Informasi Project Management

Sistem Informasi Project Management (Study Kasus PT Pilar Cipta Solusi Integratika) merupakan project penilitian tugas akhir saya sekaligus tempat magang saya di salah satu software house yang ada di Jogja. Project ini merupakan sistem untuk membantu pencatatan dan monitoring oleh project manager dan juga stakeholder dalam menghandle beberapa project yang berlangsung. Sistem ini terdapat 3 role yaitu Admin, Project Manager (PM), dan Anggota. Project ini menggunakan teknologi Laravel serta MySql sebagai databasenya.
Link Github

VieStar App

VieStar meruapakan aplikasi pembelajaran berbasis subtitle interaktif pada film akan membuat belajar bahasa Inggris lebih mudah dipahami dan lebih menyenangkan. Pengguna dapat belajar bahasa Inggris dari film, video, dan bentuk komunikasi lainnya dengan mengklik bagian mana pun dari subtitle untuk mendapatkan pemahaman yang lebih dalam tentang tata bahasa, pola kalimat, dan kosa kata. Project ini merupakan Capstone Project Product Base pada program Bangkit 2023. Dibangun menggunakan teknologi Kotlin dan Android Studio.
Link Github

Sistem Jobfair

Sistem Jobfair merupakan aplikasi untuk melakukan bursa kerja yang akan mempertemukan para pemberi kerja atau perusahaan dengan para pencari kerja. Project ini berasal dari salah satu pemkab yang ada di jogja yang biasa nya menyelenggarakan event jobfair setiap tahunnya. Dalam project ini saya sebagai front end developer dari sistem jobfair ini. Project ini menggunakan teknologi Vue JS sebagai front end nya dan Express JS dan Codeigniter sebagai back end nya.

Hilling Spot

Hilling Spot merupakan platform untuk menemukan tempat hiling atau wisata yang ada di sekitar kita. Project ini saya buat untuk mengikuti lomba OLIVIA (Olimpiade Vokasi Indonesia) bidang design web yang diprakarsai oleh Forum Pendidikan Tinggi Vokasi Indonesia (FPTI). Tahun 2022, Universitas Negeri Yogyakarta. Project ini menggunakan teknologi Vue JS dan menggunakan API dari mapbox dan openstreetmap sebagai API map nya.
